##############################################
  Scoring System:
##############################################

Each cell in the shortest path is worth 5 points. Every other cell is worth -1 points.
Toggling the path to completion subtracts 10 points from your score. Toggling a hint
subtracts 3 points from your score. The first time a cell is visited is the only time
the cell's score is added to your total.


##############################################
  Hint Option:
##############################################

The hint option helps the user out by displaying the next unvisited cell in the shortest
path so they can figure out how to get there.
